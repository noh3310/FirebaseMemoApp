# firebase_memo_app
- 메모를 생성하고 공유할 수 있는 앱

## Feature
- 메모 생성
- 메모 공유
- 로그인, 회원가입(공유할 때 회원가입 필요)

## 프로젝트 목표
- Firebase를 사용해 외부 사용자와 공유할 수 있도록 구현
- RxDart 라이브러리 사용
- Bloc 아키텍처 적용
